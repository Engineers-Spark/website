# Gallery Directory

Add your images to this directory and they will automatically appear in the gallery page.

## Supported formats:

- JPG, JPEG
- PNG
- GIF
- WebP

## Naming suggestions:

- `image1.jpg`, `image2.png`, etc.
- `photo1.jpg`, `photo2.png`, etc.
- `team1.jpg`, `event1.png`, etc.
- `workshop1.jpg`, `ctf1.png`, etc.

The gallery page will automatically scan for images with these naming patterns and display them.
